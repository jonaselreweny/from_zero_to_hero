# From Zero to Hero

[![Open in GitHub Codespaces](https://github.com/codespaces/badge.svg)](https://codespaces.new/jonaselreweny/from_zero_to_hero)

This repository contains the workshop modules for the Neo4j "From Zero to Hero" session.

All step-by-step instructions are in the exercises in the `modules` folder.

## Before You Start

1. Open this repository in GitHub Codespaces.
2. Create a Neo4j Aura Free instance and make sure you save the connection details (URI, database, username, password).

## Workshop Modules

1. `modules/1.load_data.ipynb`
2. `modules/2.query_graph.cypher`
3. `modules/3.vector_index_search.ipynb`
4. `modules/4.neo4j_mcp.ipynb`
5. `modules/5.build_your_own_mcp_server.ipynb`

Work through the modules in order.
