# Prerequisites

* Internet connection (for accessing GitHub Codespaces and Neo4j Aura Free)
* Neo4j Aura Free instance which is empty and ready to be used for the workshop
* GitHub account to use Codespaces

## Setting Up Neo4j Aura Free

1. Go to Aura Free product page and click "Start Free" to create an instance. https://neo4j.com/product/auradb/
2. Use an existing account or create an account with your email address and a password.
3. Fill in the form to create a free instance.
4. Download the admin credentials and save them in a secure location. You will need these credentials to connect to your Neo4j instance from the workshop code.

## Setting Up GitHub account

1. Ensure you have a GitHub account. If you don't have one, sign up at https://github.com/
